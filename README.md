# boilerplate-solidity(buidler & ethers)

A boilerplate for Solidity project.

## Setup

## Commands

- Setup buidler project

```bash
npx buidler
```

- Test

```bash
npx buidler test
```

- Linter

```bash
npx buidler check
```
